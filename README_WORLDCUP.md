# תחרות ניחושי מונדיאל 2026

אתר תחרות ניחושים מקצועי לשלב הבתים של FIFA World Cup 2026.

## תכונות

### משתמש
- **4 טורנירים**: ₪200, ₪500, ₪1000, ₪2000
- **72 משחקי שלב הבתים** – לוח משחקים מלא ומעודכן (USA Today, דצמבר 2025)
- **ניחושים 1/X/2** – ניצחון בית / תיקו / ניצחון חוץ
- **שיטת ניקוד**: כל תוצאה נכונה (ניצחון או תיקו) = 3 נקודות. טעות = 0.
- **דירוג** – לפי טורניר, רק טפסים שאושרו
- **צפייה בטפסים** – כל הטפסים גלויים (סטטוס: ממתין/אושר/נדחה)

### מנהל
- טפסים, משחקים, דירוגים, כספים (שקיפות), סוכנים, כל השחקנים, הגדרות (טורנירים/נעילה/מחיקה)

## הרצה

```bash
npm install
npm run dev
```

האתר רץ עם SQLite מקומי (`./data/worldcup.db`).

## בדיקה לפני פרסום

- `npm run check` – בדיקת TypeScript
- `npm run build` – בנייה ל-`dist/`
- `npm run start` – הרצת פרודקשן
- אופציונלי ב-`.env`: `JWT_SECRET`, `ADMIN_SECRET`, `AGENT_COMMISSION_PERCENT_OF_FEE`

## משתמש Admin

הגדר ב-`.env`:
```
OWNER_OPEN_ID=local-USERNAME-TIMESTAMP
```

או עדכן ישירות ב-DB את ה-role של המשתמש ל-admin.
